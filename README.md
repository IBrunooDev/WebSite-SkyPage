## 📋 Sobre
Site institucional da SkyPage, desenvolvido com HTML, CSS e JavaScript puro. Um projeto simples e elegante para apresentar os serviços da empresa.

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)

## 📁 Estrutura do Projeto
#### ✨ Funcionalidades
- Menu responsivo
- Animações simples
- Navegação suave
- Cards de serviços
